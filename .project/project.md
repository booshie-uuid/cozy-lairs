## Version Numbering

V{plan}.{task}.{build}

- **Plan**: Incremented each time an implementation plan is created.
- **Task**: Incremented each time a task within an implementation plan is completed. Resets to zero when plan changes.
- **Build**: Incremented each time project is built or packaged for deployment.
