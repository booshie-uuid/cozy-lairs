# Icon Sources

Decorative SVGs used by the cozy theme. All hand-drawn for Cozy Lairs and licensed under the project's CC BY-SA 4.0.

| File | Use | Notes |
| --- | --- | --- |
| `corner.svg` | Top-left corner flourish on toast panels and the min-viewport overlay. CSS rotates it for the other three corners. | Hand-drawn geometric curl + dot; gold fill (`#f0c674`). |
| `divider.svg` | Horizontal rule with a central diamond dingbat. Used as a section break in the min-viewport overlay. | Hand-drawn; thin gold rules + diamond fill. |
| Inline minion-with-candle sketch | Lives in `index.html` inside `#loading-overlay-inner` so it animates with the overlay's fade-out. | Hand-drawn; simple silhouette + flame shape. |

If the project later wants more elaborate ornaments, [Game-icons.net](https://game-icons.net/) (CC BY 3.0) is a good source — record any imports here with their original ids and a note on the licensing terms.
